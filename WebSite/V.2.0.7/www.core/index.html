Silver Sound Detector

Silver Sound Detector est une application développée par Mister Papaye,
qui récupère le son du micro en décibels et affiche
des alertes lorsque certains paliers sont atteints.

Silver Sound Detector, codée en JavaScript, HTML et CSS,
est basée sur Electron et Nodejs. Cette application,
comme la plupart des applications SilverCore, est
open-source sur son GitHub accesible avec le boutton
Ci-dessous.

Les Fondateurs

Mister Papaye

Co-fondateur de SilverCore ;
Développeur en HTML, CSS et JavaScript.
Gestionnaire d'un petit data center.

Mister Papaye, co-fondateur de SilverCore.
Passionné par la technologie,
il se lance dans le développement web dès son jeune âge.
Il prend ensuite une grande pause dans le développement
avant de reprendre pour s'occuper du développement
des sites web d'une association.

En 2024, il rencontre Jemy5 et lance ensemble
le projet d'un serveur Minecraft, Silverdium,
qui, de simple projet entre amis,
prend rapidement de l'ampleur pour devenir
un grand projet public. Peu de temps après,
ils lancent SilverCore affain gérer
plusieurs projets Silver avec une bonne éthique
et transparence envers le client.
