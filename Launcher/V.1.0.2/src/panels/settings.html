Paramètre de votre RAM

Définir la RAM alloué à votre jeu,

Silverdium vous conseil les reglages suivant
selont la quantité de RAM que vôtre pc possede :
8G -> 6G | 16G -> 8G | 32G -> 16G | 64G -> 16G

A noter qu'il faut éviter de mettre une trop grande quantité sinon cela peut provoquer des pertes de performances sur vos autres applications.

-RAM Maximale: Indique la quantité totale de RAM allouée à votre jeu. Il est recommandé de ne pas dépasser 50% de la quantité totale de RAM présente sur votre PC.

-RAM Minimale: Indique la quantité minimale de RAM allouée au démarrage de votre jeu. Il est recommandé de ne pas mettre plus de 4Go pour cette valeur.

Vous possédez un total de 0 Go RAM.
Vous possédez un total de 0 Go RAM disponible.
Paramètres de resolution Minecraft

Définissez une résolution personnalisée afin d'avoir un jeu adapté à votre écran.
Par défaut le jeu démarre en 854 X 480,
vous pouvez modifier cette valeur pour un confort de jeu optimal. La résolution idéale est de 1920 x 1080 mais vôtre pc n'est paut être pas assé puissant.

Résolution recommandé : 1920 x 1080 | 1080 x 720 | 1280 x 720 | 854 x 480

Paramètre de téléchargement
Ce paramètre détermine le nombre maximum de fichiers pouvant être téléchargés simultanément lors de la mise à jour ou du téléchargement de ressources.

Veuillez noter que la modification de ce paramètre peut affecter les performances de téléchargement.
Il est recommandé de ne pas dépasser les limites recommandées par défaut,
sauf si vous rencontrez des problèmes spécifiques de connectivité.
